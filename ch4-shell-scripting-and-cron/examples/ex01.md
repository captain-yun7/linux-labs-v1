### ex02 : 쉘 스크립트 기초

#### 1. 파일 생성

```bash
# 실습 디렉토리
mkdir shell-scripts && cd shell-scripts

# 쉘 스크립트 생성
vim hello.sh
```

#### 2. 스크립트 작성

```bash
#!/bin/bash

echo "Hello, Shell !!!"
echo "Hello, script !!!"

```

#### 3. 실행 권한 부여

```bash
chmod +x myscript.sh
```

#### 4. 스크립트 실행

```bash
./myscript.sh
```
