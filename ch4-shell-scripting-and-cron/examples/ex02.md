### ex02 : 반복문

#### 1. 테스트 환경 구축

```bash
echo "테스트 파일 1" > test1.txt
echo "테스트 파일 2" > test2.txt
echo "테스트 파일 3" > test3.txt
```

#### 2. 스크립트 파일 생성

```bash
vim rename_txt_to_bak.sh
```

- 파일 내용

```bash
#!/bin/bash
# 모든 .txt 파일을 .bak.txt로 변경

for file in *.txt; do
    mv "$file" "${file%.txt}.bak.txt"
done
```

#### 3. 실행 권한 부여

```bash
chmod +x rename_txt_to_bak.sh
```

#### 4. 스크립트 실행

```bash
./rename_txt_to_bak.sh
```
